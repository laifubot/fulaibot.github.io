# 电脑模拟器端使用

电脑模拟器，指的是在PC电脑上的安卓模拟器中使用，安卓模拟器在电脑中模拟了一个安卓手机系统环境，可以运行安卓应用程序，此种使用方法需要在电脑上设置

# 下载安装安卓模拟器

如果您还没有在电脑上安装安卓模拟器，推荐使用雷电模拟器点击下载雷电安卓模拟器 (opens new window)

TIP

雷电模拟器只能用上面的链接下载4.0.83版本的，安装完成后不要升级模拟器版本，其他版本可能会打不开游戏
也可能尝试使用其他模拟器，不过有可能打不开游戏

# 修改模拟器设置

模拟器安装完之后打开，将模拟器分辨率修改为 手机版720*1280 320dpi， 确认开启了root权限
修改设置之后重启一下模拟器

# 下载安装游戏到模拟器中

点击下载游戏 (opens new window)

TIP

如果是在电脑上下载，下载完成后将安装包拖拽进模拟器中进行安装

# 下载脚本到模拟器中

点击下载脚本 (opens new window)

TIP

如果是在电脑上下载，下载完成后将安装包拖拽进模拟器中进行安装

# 登录游戏

打开游戏登录自己账号就可以了

# 运行脚本

点击模拟器中的福来马娘助手图标脚本就打开了，此时并没有运行起脚本，而是会开启脚本的悬浮按钮：
点击悬浮按钮后会显示一圈按钮，点击绿色三角形按钮即可开始运行脚本，运行脚本后会显示配置界面，可以设置基本配置后点击开始运行脚本
