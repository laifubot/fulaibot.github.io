# 助手养马设置说明

# 功能设置

功能设置页面可以选择助手要执行的功能，勾选的功能会按照内置顺序执行，勾选领取奖励和竞技场比赛后，每次育成结束后会在主页面检查是否可以领取奖励和进行竞技场比赛，除育成功能外的其他功能详细设置在勾选此功能后可以在“其他”页面进行设置

# 育成设置

育成设置页面可以进行详细的育成设置，例如选择要育成的马娘和继承的马娘等，也可以选择设置赛程，设置育成事件的指定选择等

# 赛程设置

在育成页面下半部分有赛程下拉选择：

选项1和2分别对应游戏内置推荐赛程，选择选项3则会出现自定义赛程设置按钮，点击选择按钮则会进入赛程选择页面：

在赛程选择页面即可自定义选择赛程

# 育成事件设置

助手内内置了大部分育成事件，且遇到了不认识的事件会自动上传，随后我们会在下次更新时支持此事件的识别，育成时默认助手会根据育成目标自动进行事件的选择，但是如果您想要指定一些事件的选项也是可以的，在育成页面最下面有事件选择设置，可以在事件下拉框中搜索选择事件，然后指定此事件选择点击确认即可

# 属性技能设置

属性技能页面可以设置育成目标属性和育成技能学习等

# 其他设置

“其他”页面，在勾选育成之外的功能后，此页面会显示此功能的详细设置，此外还可以设置微信通知、保存读取设置存档、填入激活码、反馈意见等
