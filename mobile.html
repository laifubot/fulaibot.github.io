# 手机端使用

手机上使用脚本，目前只支持在安卓手机上使用，苹果手机用户可以将账号迁移到安卓手机上使用，或者在电脑上使用，或者可以通过云手机（需另外购买云手机）可以在苹果手机上挂机 手机上使用需要在手机上下载虚拟机，可以实现后台挂机

# 在手机上下载虚拟机

这里提供了破解版的 VMOS Pro： 点击跳转下载虚拟机 (opens new window) 密码：888 下载完之后在手机上安装即可

TIP

如果下载失败了，请换个浏览器下载
如果提示病毒，请放心安装破解版会误报病毒

# 设置虚拟机

安装完成后，打开VMOS虚拟机，给虚拟机需要的所有权限
1、
选择我是老鸟
2、点击添加虚拟机，选择下载并添加安卓7.1极客版，打开虚拟机后点击 设置-虚拟按键-关闭按键-

3、打开vmos设置页面，打开超级用户权限 和 Xposed框架，如果账户是谷歌则还要打开谷歌框架

TIP

如果打开权限时提示要会员，则后台关闭VMOS重新启动后再打开权限即可

4、在设置页面中点击显示设置，在显示设置中关闭虚拟按键，并添加并修改分辨率为720x1280，320DPI 修改后需要重启虚拟机
5、在虚拟机桌面中点击桌面底部的添加按钮，添加赛马娘游戏
6、添加完游戏后无法直接打开，因为游戏检测了Root环境，所以需要再安装个隐藏Root app才可以正常打开游戏，在虚拟机中的浏览器中打开此链接下载并安装隐藏Root app：https://wwxc.lanzouo.com/iRVDO0jevy3c (opens new window) 密码：888
7、安装完隐藏Root后中在虚拟机中打开Xposed，在模块设置中勾选隐藏Root

8、然后打开隐藏Root，点击管理应用，点击右上角➕，下滑找到赛马娘点击即可，然后重启虚拟机，就可以正常打开赛马娘游戏登录账号下载更新了

# 在虚拟机中安装脚本

点击下载脚本 (opens new window)，可以安装到手机上后导入虚拟机，也可以直接中虚拟机中打开下载链接下载安装

# 登录游戏

在虚拟机中登录自己的游戏账号，下载更新游戏资源

# 运行脚本

点击虚拟机中的福来马娘助手图标脚本就打开了，此时并没有运行起脚本，而是会开启脚本的悬浮按钮：
点击悬浮按钮后会显示一圈按钮，点击绿色三角形按钮即可开始运行脚本，运行脚本后会显示配置界面，可以设置基本配置后点击开始运行脚本
