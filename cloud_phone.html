# 云手机上脚本使用

云手机是在云端服务器上模拟的一个安卓手机环境，可以在上面挂机手游而不占用手机的资源，云手机客户端可以将云手机的画面传输给自己的手机上而监控云手机的运行状况

TIP

云手机是收费的

# 下载购买云手机

目前市场上有很多云手机品牌可以自行选择，这里推荐多多云和红手指：
多多云云手机 (opens new window)
红手指云手机 (opens new window)

# 将软件安装到云手机中

云手机一般都支持将自己手机中的软件上传到云手机中，也可以直接在云手机中下载安装
1、将游戏安装到云手机中，点击下载游戏 (opens new window) 2、点击下载脚本 (opens new window)，将脚本安装到云手机中

# 登录游戏

在云手机中登录自己的游戏账号

# 运行脚本

点击云手机中的福来马娘助手图标脚本就打开了，此时并没有运行起脚本，而是会开启脚本的悬浮按钮：
点击悬浮按钮后会显示一圈按钮，点击绿色三角形按钮即可开始运行脚本，运行脚本后会显示配置界面，可以设置基本配置后点击开始运行脚本
